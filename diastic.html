Diastic Animations and flocks Bio
A very simple diastic machine inspired by the work of Daniel Schiffman of NYU.
The result is not gibberish.
It is in fact swedish corporate jargong.
It will pick a number of words from a source text based on the input. The first words first letter
is the same as the first eletter of the input. The second word has the same second letter as the input word,
the third the same third letter and so on.
