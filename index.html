Diastic Animations and flocks Generators and API

The diameters of asteroids near earth today

Fetching data about asteroids close to earth from NASA. Fetching JSON data, nested in several layers, with a variable that automatically requests data between the current date and a week ago.

Data-set: All the data is from the NASA JPL Asteroid team (http://neo.jpl.nasa.gov/).
